Flow Breaker Shield Privacy Policy

Effective date: April 2, 2026

Single purpose

Flow Breaker Shield helps users avoid adult-content browsing and protect focus by blocking websites, auto-detecting clearly adult pages, learning high-risk browsing hours locally, and showing recovery-oriented reminders.

What data the extension handles

  • Web history signals such as blocked domain names, page URLs/referrers involved in blocked redirects, and timestamps of blocked attempts.
  • Website content signals such as page title, meta description, meta keywords, URL path, and visible page text used locally to detect clearly adult pages.
  • User activity signals inside the extension such as Emergency Shield button presses and local reminder state.

How data is used

Data is used only to provide the extension's blocking, auto-detection, risk-hour learning, and reminder features.

Where data is stored

Data is stored locally in Chrome extension storage on the user's device. Flow Breaker Shield does not send this data to an external server.

Data sharing and sale

Flow Breaker Shield does not sell user data and does not transfer user data to third parties, except where required by law.

Remote code

The extension does not use remote JavaScript or remote WebAssembly. All extension code is packaged in the extension bundle.
